# Solar Forecasting Project

This project focuses on solar forecasting using multiple deep learning and baseline models. It includes data cleaning, model training, evaluation, diagnostics, and multi-horizon forecasting outputs.

## Project Structure

    solar_forecasting_project/
    ├── README.md
    ├── code/
    ├── graphs/
    └── results/

## Code
The `code/` folder contains Python scripts for:
- data cleaning
- baseline models
- BiLSTM model
- CNN-LSTM model
- diagnostics
- evaluation
- multi-horizon forecasting

## Graphs
The `graphs/` folder contains plots for:
- training loss
- prediction results
- baseline plots
- horizon forecast demo
- horizon-wise R2
- horizon-wise RMSE

## Results
The `results/` folder contains:
- model outputs
- evaluation summaries
- diagnostics
- multi-horizon summaries

## Models Used
- Linear Regression baseline
- Baseline LSTM
- BiLSTM
- CNN-LSTM
